# 🐍 贪吃蛇游戏

一个功能完整、界面美观的贪吃蛇小游戏，使用原生 HTML5 + CSS3 + JavaScript 开发。

## ✨ 功能特点

### 核心功能
- 🎮 **经典玩法**: 控制蛇移动、吃食物、增长身体
- 🎯 **碰撞检测**: 撞墙或撞到自己身体时游戏结束
- 🍎 **随机食物**: 食物随机出现在游戏区域内
- 📊 **分数系统**: 实时显示当前分数和历史最高分

### 游戏控制
- ⌨️ **双键盘支持**: 支持方向键（↑↓←→）和 WASD 键控制
- ▶️ **开始/暂停**: 随时开始、暂停或重新开始游戏
- ⚡ **难度选择**: 三种难度等级（简单/中等/困难）
- ⏸️ **空格键暂停**: 按空格键快速暂停/继续

### 用户体验
- 📱 **响应式设计**: 完美适配不同屏幕尺寸（电脑、平板、手机）
- 🎨 **现代化UI**: 渐变色彩、圆角设计、流畅动画
- 💾 **本地存储**: 自动保存最高分记录
- 👁️ **视觉反馈**: 蛇头有眼睛、食物有渐变效果、网格背景

## 🚀 如何运行

### 方法一：直接打开（推荐）
1. 下载或克隆本项目到本地
2. 双击打开 `index.html` 文件
3. 浏览器会自动运行游戏

### 方法二：使用本地服务器
```bash
# 使用 Python 3
python -m http.server 8000

# 或使用 Python 2
python -m SimpleHTTPServer 8000

# 或使用 Node.js (需要先安装 http-server)
npm install -g http-server
http-server

# 或使用 PHP
php -S localhost:8000
```

然后在浏览器中打开：`http://localhost:8000`

## 🎮 游戏操作

### 键盘控制
- **方向键**: ↑ ↓ ← → 控制蛇的移动方向
- **WASD 键**: W (上) / S (下) / A (左) / D (右)
- **空格键**: 暂停/继续游戏

### 按钮控制
- **开始游戏**: 开始新游戏
- **暂停**: 暂停当前游戏
- **重置**: 重置游戏状态
- **难度选择**: 在游戏开始前选择难度

### 游戏规则
1. 蛇会朝着当前方向持续移动
2. 吃到红色食物后，蛇身会增长一节，分数 +1
3. 撞到墙壁或自己的身体时，游戏结束
4. 蛇不能立即反向移动（例如向右移动时不能直接向左）

## 🛠️ 技术实现

### 技术栈
- **HTML5**: 页面结构、Canvas 画布
- **CSS3**: 样式设计、响应式布局、动画效果
- **JavaScript**: 游戏逻辑、事件处理、本地存储

### 核心技术点
- **Canvas API**: 绘制游戏画面（网格、蛇、食物）
- **LocalStorage API**: 保存最高分记录
- **定时器**: 使用 `setInterval` 控制游戏循环
- **键盘事件**: 监听方向键和WASD按键
- **响应式设计**: CSS媒体查询适配不同屏幕

### 代码特点
- ✅ 代码结构清晰，模块化设计
- ✅ 变量命名规范，易于理解
- ✅ 包含必要的代码注释
- ✅ 无外部依赖，纯原生实现
- ✅ 性能优化，流畅运行

## 🎯 难度等级说明

- **简单 (慢速)**: 120ms/帧 - 适合新手玩家
- **中等 (正常)**: 80ms/帧 - 标准难度
- **困难 (快速)**: 50ms/帧 - 挑战高手

## 💡 游戏技巧

1. **提前规划路线**: 不要急于吃食物，先确保有退路
2. **利用边缘**: 靠近边缘移动可以减少被困的风险
3. **螺旋移动**: 使用螺旋移动模式可以更安全地控制空间
4. **避免急转**: 在高速模式下，尽量避免连续急转弯
